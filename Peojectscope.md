# Project Scope

## Problem
People often do not have their resume readily available when networking at events, career fairs, or unexpected professional meetings. Searching through files, emails, or cloud storage in the moment can be inconvenient and unprofessional.

## Solution
Build a web application where users enter their professional details through a form, and the system generates a polished PDF profile/resume. The app also creates a wallet-friendly digital card with a QR code that links to the generated PDF, making it easier to access and share quickly.

## Target Users
- University students
- Job seekers
- Professionals attending networking events
- Anyone who wants quick access to a shareable professional profile

## Inputs
The user will enter their information through a form, including:
- Full name
- Professional title or headline
- Contact information
- Education
- Work experience
- Skills
- Links such as LinkedIn, portfolio, or GitHub

## Outputs
The system will generate:
- A downloadable PDF profile/resume
- A QR code linked to the generated PDF
- A wallet-style digital card for quick access
- A shareable link to the created profile PDF

## Features for V1
- Form to collect professional details
- Generate a PDF from user input
- Generate a QR code linked to the PDF
- Display a digital card preview with essential profile details
- Allow the user to download the PDF

## Non-Goals for V1
- Full LinkedIn profile import
- Automatic LinkedIn scraping
- Mobile app development
- User authentication and accounts
- Hosting a complete personal portfolio website
- Advanced resume customization/themes

## Future Versions
- Convert the generated profile into a static personal website
- Add Apple Wallet pass support
- Support LinkedIn PDF upload and parsing
- Offer multiple resume/card templates
- Allow users to update and save multiple profile versions
- Build a mobile app version
